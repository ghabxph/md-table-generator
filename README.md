# md-table-generator
Markdown Table Generator. This is an experimental project. I am going to use this tool for personal use. Some time later,
this program shall evolve into a useful stuff that is worth to share to all.

## Usage
```
./script.py 10 3 5
```

### Creates
```
|     |     |     |
|-----|-----|-----|
|     |     |     |
|     |     |     |
|     |     |     |
|     |     |     |
|     |     |     |
|     |     |     |
|     |     |     |
|     |     |     |
|     |     |     |
```
